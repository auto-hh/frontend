# Auto hh

Приложение для соискателей, интегрированное с hh.ru. Автоматический поиск наиболее подходящих вакансий на основе вашего резюме.

## 🚀 Быстрый старт

### Требования

- Docker и docker compose
- 80 порт должен быть открыт на локальной машине

### Запуск

```bash
docker compose up -d --build
```

## 🔐 Авторизация

### Как работает

1. Пользователь нажимает «Войти через hh.ru»
2. Перенаправление на OAuth-провайдер
3. После успеха — редирект обратно с кукой `auth=true`
4. Приложение проверяет куку и загружает профиль
5. Открывается доступ к защищенным страницам

### Хранение состояния

| Данные             | Где хранится   | Инструмент           |
| ------------------ | -------------- | -------------------- |
| Статус авторизации | Cookie         | `entities/auth`      |
| Вакансии           | TanStack Query | `entities/vacancies` |
